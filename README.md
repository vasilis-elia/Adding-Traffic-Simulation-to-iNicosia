# Adding Traffic Simulation to the Digital Twin of Nicosia (iNicosia)
Bachelor thesis Project

Requirements
Unity 6000.3.11f1 or later
Python 3.12 or later
Eclipse SUMO 1.26.0
Cesium for Unity package
Cesium ion account and access token

Simulation files for running a specific area of Nicosia are added.

To run SUMO simulation on SUMO GUI run the sumo_unity.py file.
